# Aetheris

> **A GPU-native scientific simulation engine and professional editor built with modern C++.**

Aetheris is an ongoing project focused on building a high-performance **2D scientific simulation platform** that combines GPU computing, real-time visualization, and a modern editor into a single extensible environment.

Unlike traditional game engines, Aetheris is designed for **interactive simulation, experimentation, and future AI-assisted workflows** within a mathematically accurate Cartesian world.

---

## Features

### Current

* Modern C++ engine architecture
* Professional editor with custom UI
* Interactive Cartesian workspace
* Real-time particle simulation
* Physics inspector & simulation controls
* Dockable editor interface
* Modular subsystem architecture
* CPU reference simulation

### Planned

* GPU Compute Shader simulation
* Scene hierarchy & asset browser
* Material editor
* AI-powered simulation assistant
* Performance profiler
* Timeline & graph editor
* Hand tracking & voice interaction
* Plugin system

---

## Technology Stack

* **Language:** C++20
* **Graphics API:** OpenGL 4.3+
* **GPU Compute:** Compute Shaders (planned/in progress)
* **Framework:** Raylib
* **GUI:** Dear ImGui
* **Build System:** CMake
